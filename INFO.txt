Bienvenido a mi repositorio para la asignatura de Análisis Numérico.

La estructura del repositorio consiste en lo siguiente:
> Existe una carpeta por cada Método Numérico
> Cada carpeta contiene un código fuente, imágenes del procesamiento del código, un archivo README.txt
> El lenguaje utilizado es Python, el cuál se podrá descargar de la siguiente dirección:
		http://www.python.org/download/
  la versión implementada es 2.7, existe una herramienta 2to3 para poder pasar código de la versión 2.7
  a la versión 3.3.
> Si es requerido descargar todo el repositorio favor de preguntar, se implementa GIT desde la terminal.
> El lenguaje utilizado para resolver sistemas de ecuaciones es Java.
> Java es un lenguaje de programación compilado/interpretado, por lo que se requiere un compilar que se
  puede descargar de la página oficial:
		https://www.java.com/es/download/
> Para poder compilar el código Java se requiere ejecutar la siguiente instrucción desde la temrinal:
		javac nombre_del_archivo.java
> Una vez que se generé el archivo compilado, es decir, que se tenga:
		nombre_del_archivo.class
  Se procede a ejecutar el código con el siguiente comando
		java nombre_del_archivo
  Como se puede observar no requiere indicar la extensión .class, ya que la Java Virtual Machine lo
  identifica por default.
